Particle System
===============

Particle System - physics particle simulator based on JavaFX.

## How to run
1. Clone the repo
2. Enter to the directory
3. Run following command (depends on your OS)
```
sudo chmod +x ./gradlew && ./gradlew run  # Unix-like
.\gradlew.bat run                         # Windows 
```

5. PROFIT`!!!`
